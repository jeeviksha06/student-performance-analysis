# Student Performance Analysis

## Project Overview
A complete data analysis project that analyzes student 
performance across subjects using Python and Pandas,
with visualizations created using Matplotlib.

## Objectives
- Load and explore student performance data
- Check and handle missing values
- Calculate key performance metrics
- Create visualizations (bar chart, pie chart)
- Generate meaningful insights from data

## Setup & Installation
1. Install Python 3.x
2. Install required libraries:
pip install pandas matplotlib
3. Clone this repository
4. Run: python main.py

## Project Structure
- `main.py` — main analysis script
- `data/` — contains dataset
- `visualizations/` — contains generated charts
- `report/` — contains written insights

## Charts Generated
1. Bar Chart — Average score comparison by subject
2. Pie Chart — Grade distribution among students

## Key Findings
- Science has highest average score (78.10)
- 43.3% students scored Grade A
- Average attendance is 86.87%
- Top student: Aaron
